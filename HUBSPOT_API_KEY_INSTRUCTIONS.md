# How to get your HubSpot API key

1. **Log in to your HubSpot account.**
2. **Navigate to Settings > Integrations > API key.**
3. **If you have an existing key, you can copy it from here. Otherwise, click "Create key" to generate a new one.**
4. **Once you have your API key, open the `hopehands/settings.py` file in the project.**
5. **Find the line that says `HUBSPOT_API_KEY = 'your-hubspot-api-key'` and replace `'your-hubspot-api-key'` with your actual API key.**
6. **Save the file.**

Your application should now be able to connect to the HubSpot API.
